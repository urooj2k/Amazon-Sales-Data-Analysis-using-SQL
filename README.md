# Amazon Sales Data Analysis using SQL

## Introduction

This project analyzes an Amazon product dataset using SQL to uncover insights about product pricing, discounts, ratings, and customer reviews. The goal is to understand product trends, customer satisfaction, and pricing strategies through structured queries and data analysis.

## Data Components

The dataset contains information about Amazon products, pricing, and customer feedback. Key fields include:

* **product_id** – Unique identifier for each product
* **product_name** – Name of the product
* **category** – Product category
* **actual_price** – Original price before discount
* **discounted_price** – Price after discount
* **discount_percentage** – Percentage discount applied
* **rating** – Average customer rating
* **rating_count** – Total number of ratings
* **review_title** – Title of the customer review
* **review_content** – Detailed customer feedback
* **user_id** – Unique identifier for the reviewer

These fields help analyze product performance, discount trends, and customer sentiment.

## Conclusion

Using SQL queries, this project extracts meaningful insights from the Amazon sales dataset. The analysis highlights pricing patterns, identifies highly discounted and highly rated products, and explores customer feedback trends. These insights can help businesses and sellers make informed decisions about pricing strategies, product quality, and customer satisfaction.
